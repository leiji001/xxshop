## 快速开始

1. 先阅读 **[API-总览与通用规范.md](./API-总览与通用规范.md)**，了解全局约定
2. 根据你的模块选择对应的文档文件
3. 接口概览表在总览文档的第 九 节

## 生成说明

本文档通过逐行阅读源代码生成，所有字段名、参数名、响应结构均与代码保持完全一致。不一致之处已在文档中以 ⚠️ 标注或在总览文档中列出。
